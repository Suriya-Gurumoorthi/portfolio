About the Project

  • I developed an innovative Trip Itinerary Generator that transforms travel planning into a seamless and personalized experience.
  • This tool takes user inputs—such as starting location, destination, preferred mode of transportation, budget, food preferences, and trip duration—and crafts a detailed, day-by-day itinerary tailored to their needs.
  • Key Features:
    • 🔵 Provides weather predictions for the travel dates, helping users pack and plan activities with confidence.
    • 🔵 Delivers approximate flight ticket prices, enabling budget-conscious travelers to make informed decisions.
    • 🔵 Offers currency exchange rate approximations, simplifying financial planning across borders.
  • This project showcases my ability to blend API integration, data processing, and user-centric design to solve real-world problems—perfect for anyone dreaming of their next adventure!
