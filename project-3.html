About the Project

  • This project focuses on predicting whether a customer will book a flight ticket based on historical booking patterns and contextual variables. Using booking data from British Airways, the model analyzes a wide range of factors such as trip duration, number of passengers, flight day, booking source, meal preferences, and other behavioral variables to forecast booking decisions.
  • The dataset was initially messy, with significant amounts of missing values, inconsistencies, and outliers. A major portion (around 60% of the total time) was dedicated to data cleaning, imputation, and outlier handling — turning the raw data into a structured, model-ready format.
  • After preprocessing, multiple machine learning models were trained and optimized using hyperparameter tuning techniques to adapt them specifically to the problem space and got Random Forest as the lead performing model with 85% Precision Score.
  • This project showcases my ability to handle effective data cleaning, GridSearchCV vs RandomSearchCV and proper feature engineering to prevent the model from overfitting.
